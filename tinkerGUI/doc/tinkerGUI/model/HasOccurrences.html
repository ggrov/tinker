HasOccurrences

trait HasOccurrences extends AnyRef

Tactic behaviour for having occurrences

An occurrence represents the fact that a node on a graph is linked with a tactic. It is a tuple with one string representing the graph in which the occurrence is, one integer for the graph id, and one string being the node id.
